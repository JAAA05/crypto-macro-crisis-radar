# Example Outputs

This folder contains lightweight sample outputs generated from the project pipeline. They are included for portfolio review so a reader can understand what the system produces without running the full research suite.

Key files:

- `reports/latest_market_regime_report.md` — latest model-generated market regime report.
- `research/crypto_macro_research_dashboard.md` — portfolio-style research dashboard summary.
- `research/event_study_report.md` — event-study summary.
- `research/lag_shock_report.md` — lag and shock analysis.
- `research/var_lite_report.md` — VAR Lite econometric analysis.
- `research/var_robustness_report.md` — VAR robustness checks.
- `research/ablation_v2_report.md` — walk-forward ablation study.
